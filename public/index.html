Set the Wisp server (proxy base URL) here. Then enter a site or search terms and launch.

If blank, it uses this server.
A “Wisp server” here means “any deployed proxy with the same endpoints”: /proxy?url= and /proxy?q=.
Tips
  • Type a normal URL (with or without https://).
  • Type words/spaces to search (it will use Google automatically).
  • Top buttons work everywhere: Back/Forward/Reload/Home.
  • If a site breaks login, it may be using anti-bot checks — but this proxy is tuned to work on more sites than the older version.
